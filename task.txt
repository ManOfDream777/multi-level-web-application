1. Multi-level web application with authentication
Description: An application that includes registration, login, authorization with different access levels (for example, admin, user).
Technologies: Python (Django/Flask), React, PostgreSQL.
Functional:
Registration and login using tokens (JWT).
Different roles of users (admin can manage content, users can create or edit only their own records).
REST API for interaction between frontend and backend.
